# Exercie : Collections

Concevez un programme qui calcule combien de fois chaque mot apparaît
dans une chaîne saisie par l'utilisateur·rice. Par exemple, étant donné
la saisie « Le chat noir voit le chat blanc et le chat noir suit le chat
blanc. », votre programme doit afficher le résultat suivant :

```
le : 4
chat : 4
noir : 2
voit : 1
blanc : 2
et : 1
suit : 1
```

Notez que le résultat est insensible à la casse, et que la ponctuation
est ignorée.

La méthode [split] vous sera utile.

[split]: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/split
