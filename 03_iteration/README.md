# Exercice : Itération

Implémentez les fonctions ci-dessous. N'oubliez pas de valider votre
code avec des tests automatisés.

- Une fonction `sum` qui calcule la somme d'un tableau de nombres.
- Une fonction `average` qui calcule la moyenne d'un tableau de nombres
- Une fonction `areAllSame` qui détermine si tous les nombres d'un
  tableau sont identiques.
- Une fonction `isSortedAsc` qui détermine si les nombres d'un tableau
  sont en ordre croissant.
- Une fonction `isSortedDesc` qui détermine si les nombres d'un tableau
  sont en ordre décroissant.
- Une fonction `uniques` qui retourne les nombres uniques contenus dans
  un tableau de nombres.
- Une fonction `areTwoSame` qui détermine si un tableau de nombres
  contient au moins deux fois le même nombre.
- Une fonction `areUniques` qui détermine si tous les nombres d'un
  tableau sont uniques.
