# Exercice : Événements

Concevez une fonction `createTable` qui crée un élément HTML `<table>`
à partir d'une liste de lignes. Chaque ligne est une liste de cellules,
et la première ligne contient les en-têtes. Lorsqu'on clique sur un
en-tête, les lignes du tableau sont triées par ordre alphabétique selon
cette colonne. Un second clic sur le même en-tête inverse l'ordre du tri.
