# Exercice : Modifier des éléments HTML avec JavaScript

Concevez un programme JavaScript qui affiche trois [listes déroulantes]
: une pour le jour, une pour le mois et une pour l'année. Les options
pour le jour vont de 1 à 31. Celles pour le mois vont de 1 à 12. Et
celles pour l'année vont de 1900 à l'année courante. La valeur par
défaut des trois champs correspond à la date à laquelle la page est
affichée.

[listes déroulantes]: https://developer.mozilla.org/fr/docs/Web/HTML/Reference/Elements/select
